初略安装流程 ​

  1. 安装对应版本NTQQ（特殊情况部分可能安装方式已代为安装）。
  2. 下载你需要的NapCat版本（特殊情况部分可能安装方式包含NapCat本体）。
  3. 遵照其详细教程启动NapCat本体。
  4. 进入WebUI，启动你需要的 HTTP/WS 客户端与服务端。
  5. 配置框架或者独立插件连接NapCat。

启动方式 ​

NapCat有着众多启动方式，每种方式各有优点：

  • Shell 版本：具有低内存、服务器部署简单的特点。
  • Framework 版本：具有方便人机交互、便于窥屏的特点。
